# Requirements Document

## Project Description (Input)
問い合わせ・申請フォーム機能。日本の大手小売店ポータルにおいて、海外販社（20か国以上）の担当者が問い合わせ・申請を統一フォームで送信できる機能。選択式項目（分類・緊急度・地域など）と自由記述を組み合わせ、ヘルプデスク側の分類作業負担を削減する。フェーズ1ではNext.jsフロントエンドのみのモックアップとし、モックAPIでデータを返す。関連する仮データ構造はtypes/inquiry.tsに定義予定（category/urgency/storeRegion/originalText/originalLanguage/translatedText/status/createdAt/submittedByなど）。categoryの選択肢はヘルプデスク担当者へのヒアリング後に変更される前提の仮値。
